Модуль queries - все запросы в базу,
Модуль database - подключение к базе,
Модуль csv_script - подготовка и загрузка данных в базу (бд создана, запускать не надо),
Модуль app - запуск сервера

Что нужно для запуска:
1. Клонируйте репозиторий
git clone ...
2. Создайте venv и установите зависимости
pip install -r requirements.txt
3. Поместите переданный .env файл в папку проекта
4. Запустить сервер в файле app.py
if __name__ == '__main__':
    app.run()

GET - http://127.0.0.1:5000/get_statistics - возвращает ответы на вопросы тестового задания
